Caeser Encryption

Developed a Caesar encryption program in both C++ and C-sharp to enhance data security. This project allowed for the encryption and decryption of messages using the Caesar cipher algorithm. Implemented features included customizable shift values, ensuring flexibility in the encryption process.

Morse Code Player

I have created the Morse Code Player using C-sharp and XAML, a powerful combination of technologies that brings this innovative application to life. With meticulous attention to detail and a user-friendly interface, this software is designed to decode and play Morse code messages, offering an immersive learning experience

Ticket Management System

Developed a ticket management system in C++ using object-oriented programming concepts. The system utilized a templated queue data structure for efficient ticket management. Implemented features such as adding new tickets, retrieving next tickets, and exiting the program. The project also involved working with external data from text files and included randomized ticket generation.
